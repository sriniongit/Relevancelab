Digital E-Learning

Relevance Lab offers a “Digital Learning Platform” based on new technology paradigms of Cloud, Analytics and Rich UI Designs. Cloud based architecture is adopted to provide for multi-tenant model and rich User Interface design based on “Responsive UI” for multi-devices compatibility. The Rich UI Layer is based on new touch friendly design and intuitive Usability models interacting with back end business services in a decoupled manner

The business services layer are all based on web services and integrate with external content sources for E-Books, Videos and other supplementary sources. The data architecture layer supports both traditional SQL data models and certain components are built with flexible No-SQL Data models. In order to support for Adaptive Learning and Adaptive assessments the embedded Analytics layer provides real time metrics and also comparative analysis with other students and their progress.
